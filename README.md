# Airflow Data Pipeline Demo

This project demonstrates an end-to-end data pipeline using Airflow, Python, and SQL.  
It includes a clean folder structure, placeholder DAGs, transformation scripts, and SQL models to showcase data engineering best practices.

---

## Project Structure

```
airflow-data-pipeline-demo/
│
├── dags/                     # Airflow DAG definitions
│   └── example_dag.py
│
├── scripts/                  # Python scripts for extract/transform
│   └── extract_data.py
│
├── sql/                      # SQL transformation models
│   └── transform_data.sql
│
└── config/                   # Config files (connections, variables)
    └── .gitkeep
```


## Features

- Airflow DAG structure with PythonOperator  
- Python ETL scripts for extraction  
- SQL transformation models  
- Modular folder layout  
- Easy to extend for real-world pipelines
